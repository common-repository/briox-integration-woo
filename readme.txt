Integrates WooCommerce with Briox

== Description ==

This plugin integrates WooCommerce with Briox. Automating your accounting and inventory.

It can create a Briox Invoice or Order based on a WooCommerce Order.

Customer data will be stored in the Briox customer database and will be automatically updated if changed in a new order.

This is a beta product - If you want to participate in the beta, send an email to hello@bjorntech.com
